## Getting Startted Project

1. Clone the repo:

   ```sh
   git@github.com:lehon25/MovieDB.git
   ```

2. Go to directory main

   ```sh
   cd main
   ```

3. Install npm package

   ```sh
   npm install
   ```

4. Run project

   ```sh
  react-native run-andorid
   ```
